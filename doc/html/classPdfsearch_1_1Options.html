void Pdfsearch::Options::reset ( )
static

Reset state for parsing command line options again.

If you want to parse command line options again, you need to either let the previous object call its destructor or call this function. Failure to comply may lead to segfault.

void Pdfsearch::Options::validate ( ) const

Validate options.

Check that mutually exclusive options are not given, either index, query, update or vacuum is given and matches < UNLIMITED_MATCHES. Other kind of option validation happens when option is used.

Exceptions
std::invalid_argumentif there's an invalid option.
