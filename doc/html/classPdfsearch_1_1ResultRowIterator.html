template<typename T >
std::unique_ptr<T> Pdfsearch::ResultRowIterator::column ( int  column)

Get value of a column in resultset.

Valid generic types are int, sqlite3_int64, double and std::string. Example:

* std::unique_ptr<int> pages(it.column<int>(0));
* std::unique_ptr<std::string> pages = std::move(it.column<std::string>(0));
*
Parameters
columnColumn index. 0 <= column < number of columns.
Returns
Value of a column in resultset as an unique_ptr. Can be nullptr if database column is nullable. In that case user has to check for it before use.
Exceptions
DatabaseErrorif there is no row ready or if index is < 0 or >= number of columns in resultset or if generic type is not the same as column's type.
